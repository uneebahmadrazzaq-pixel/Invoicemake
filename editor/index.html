FAQ

Common questions

Which file types can I upload?

The data-cleaning workflow accepts Excel and CSV files, while invoice templates and exports follow the selected supplier process.

How are unit prices and quantities calculated?

Quantity adds a randomized value to available and sold quantities. Unit price uses the configured randomized percentage of the start price.

Can I use different supplier formats?

Yes. Select the client, destination country, supplier profile, and required date range before processing the file.

Where are saved projects stored?

Saved clients, invoices, and project data remain in the current browser unless a hosted storage system is added later.
